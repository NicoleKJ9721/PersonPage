# 陈涛教授 - 现代化教师主页

基于苏州大学教师主页系统的美化版本，采用现代化设计理念和技术栈。

## 🎨 设计特色

### 视觉设计
- **现代化配色方案**：采用苏州大学蓝色为主色调，搭配渐变效果
- **卡片式布局**：提升信息层次感和视觉体验
- **响应式设计**：完美适配桌面、平板和移动设备
- **优雅的动画效果**：平滑过渡和微交互增强用户体验

### 功能特性
- **单页滚动设计**：流畅的页面切换体验
- **智能导航栏**：滚动时自动隐藏/显示，高亮当前章节
- **时间线展示**：教育经历采用视觉化时间线
- **标签页切换**：学术成果分类展示
- **联系表单**：集成邮件发送功能
- **返回顶部**：便捷的页面导航

## 🚀 技术实现

### 前端技术
- **HTML5 + CSS3**：语义化标签和现代样式特性
- **原生JavaScript**：无需额外框架依赖，性能优异
- **CSS Grid & Flexbox**：现代化布局方案
- **CSS变量**：便于主题定制和维护
- **Intersection Observer API**：高性能滚动监听

### 性能优化
- **图片懒加载**：提升页面加载速度
- **防抖函数**：优化滚动事件性能
- **CSS动画**：硬件加速的流畅动画
- **代码压缩**：生产环境可进一步优化

## 📝 使用说明

1. **直接部署**：将文件上传到Web服务器即可
2. **内容修改**：编辑HTML文件中的文本内容
3. **样式调整**：修改CSS变量即可改变主题色
4. **功能扩展**：在JavaScript中添加新功能

## 🎨 定制化选项

### 颜色主题
在CSS变量中修改以下值：
```css
--primary-color: #1e40af;    /* 主色调 */
--secondary-color: #3b82f6;  /* 辅助色 */
--accent-color: #f59e0b;     /* 强调色 */
```

### 字体设置
```css
--font-family: 'Inter', sans-serif;  /* 字体族 */
```

### 间距调整
```css
--spacing-md: 1rem;  /* 基础间距 */
```
